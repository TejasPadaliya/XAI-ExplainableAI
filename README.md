# Explainable AI (XAI) Projects

This repository is a growing collection of projects demonstrating various Explainable AI (XAI) techniques for machine learning interpretability and transparency.

## Overview

Explainable AI bridges the gap between complex ML models and human understanding. Each project here focuses on a specific XAI method such as LIME, SHAP, and more.
